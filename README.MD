# 💻 TechGear - E-commerce Gamer

## 📌 Sobre o Projeto

A **TechGear** é uma loja virtual focada na venda de **hardwares, periféricos e produtos voltados ao público gamer e entusiastas de tecnologia**.
O projeto foi desenvolvido com o objetivo de oferecer uma experiência moderna, rápida e responsiva para usuários que buscam equipamentos de alta performance.

## 🚀 Tecnologias Utilizadas

* **HTML5** – Estrutura das páginas
* **CSS3** – Estilização e identidade visual
* **JavaScript** – Interatividade e manipulação de dados
* **Bootstrap** – Layout responsivo e componentes prontos

## 🎮 Funcionalidades

* 🛒 Listagem de produtos
* 🔍 Sistema de filtragem por nome e categoria
* 🧺 Carrinho de compras
* ✨ Interface moderna com efeitos visuais

### 📝 Gerenciamento de Produtos (CRUD)

A página de catálogo passou a contar com funcionalidades completas de gerenciamento de produtos, permitindo:

* ➕ **Cadastrar produtos** através de formulário modal
* ✏️ **Editar produtos** existentes com atualização em tempo real
* ❌ **Remover produtos** da listagem
* 📋 **Visualizar produtos** organizados em cards responsivos
* 🔎 **Filtrar produtos** por nome e categoria

Essas funcionalidades foram implementadas utilizando manipulação do DOM com JavaScript, proporcionando uma experiência dinâmica sem necessidade de recarregar a página.

## 🎨 Interface

O design da **TechGear** foi pensado para o público gamer, trazendo:

* Cores vibrantes e contrastes marcantes
* Elementos modernos e minimalistas
* Navegação simples e intuitiva

## 📦 Objetivo

Este projeto tem como foco o desenvolvimento e prática de conceitos de front-end, como:

* Manipulação do DOM
* Organização de layouts responsivos
* Experiência do usuário (UX/UI)
* Interatividade com JavaScript
* Implementação de operações CRUD
* Utilização de componentes do Bootstrap
